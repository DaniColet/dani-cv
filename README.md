# Cloud Resume Challenge AWS

## Overview
Implementation of the Cloud Resume Challenge on AWS using:
- **Frontend**: Hugo static site (AWS Amplify)
- **Backend**: Python Lambda + DynamoDB (AWS API Gateway)
- **Infrastructure**: Terraform
- **CI/CD**: GitHub Actions

## Structure
- `frontend/`: Hugo website code
- `backend/`: Lambda function, Terraform IaC, Tests
- `docs/`: Diagram, Cost Estimation, Presentation Script
